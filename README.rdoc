= validate_email_address gem

Source code for {Getting started with Gems: How to create a gem for Rails 3}[http://bitsofgeekdom.tumblr.com/post/7443618295/] tutorial.

Extend ActiveRecord validations by checking if a string is a valid email address.

== Use
In your model:

  validates_email_address_of :email
